# ملفات الصوت للألعاب التفاعلية

هذا المجلد يحتوي على ملفات الصوت المستخدمة في الألعاب التفاعلية والاختبارات.

## الملفات المطلوبة

يرجى إضافة الملفات الصوتية التالية:

- **correct.mp3** - صوت الإجابة الصحيحة (مدة قصيرة، نغمة إيجابية)
- **incorrect.mp3** - صوت الإجابة الخاطئة (مدة قصيرة، نغمة محايدة)
- **win.mp3** - صوت الفوز/النجاح (مدة متوسطة، احتفالي)
- **time-warning.mp3** - تنبيه الوقت المتبقي (مدة قصيرة، نغمة تحذيرية)

## المواصفات المقترحة

- **الصيغة**: MP3
- **مستوى الصوت**: متوازن ومتناسق بين جميع الملفات
- **مدة الصوت**: قصيرة (0.5-2 ثانية) للصوتيات التفاعلية، متوسطة (2-5 ثواني) للفوز

## ملاحظة

إذا لم تكن الملفات موجودة، ستحاول المكونات تشغيل الصوت وتتعامل مع الأخطاء بشكل صامت بدون تعطيل التطبيق.

## مصادر مقترحة للصوتيات

- [Freesound](https://freesound.org/)
- [Zapsplat](https://www.zapsplat.com/)
